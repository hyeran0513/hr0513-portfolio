## 사용 기술 & 라이브러리

### 프론트엔드 언어 & 라이브러리

| 기술 | 설명 |
|------|------|
| **React** | 사용자 인터페이스 구축을 위한 라이브러리 |
| **Next.js** | React 기반의 프레임워크로 서버 사이드 렌더링 지원 |

### 기타 라이브러리

| 라이브러리 | 설명 |
|------------|------|
| **SCSS (SaaS)** | CSS의 확장 언어로, 변수와 믹스인 등을 지원 |
| **GSAP** | 고성능 애니메이션 라이브러리 |
| **html2canvas & jsPDF** | HTML을 캔버스로 변환 후 PDF로 저장 |
| **React Icons** | 다양한 아이콘을 React 컴포넌트로 제공 |
